# 数据提取工具使用指南

> 目录: `Notes/tools/`
> 用途：从原版游戏的 CPK 包中提取游戏数据库（GDB），并解析为 JSON 便于开发参考。

---

## 环境依赖

```bash
pip install lzokay
```

`lzokay` 是 LZO1X 解压算法的 Python 实现，与工程内 `MiniLzo.cs` 算法一致。

---

## 工具一：cpk_extract.py

解包 Softstar CPK 归档文件，移植自 [CpkArchive.cs](../../Assets/Scripts/Pal3.Core/DataReader/Cpk/CpkArchive.cs)。

### 列出 CPK 内容
```bash
python cpk_extract.py <cpk路径> --list
```

### 提取单个文件
```bash
python cpk_extract.py <cpk路径> --extract-one "cbdata\\PAL3_Softstar.gdb" 输出文件路径
```

### 全部提取（可加过滤）
```bash
python cpk_extract.py <cpk路径> 输出目录
python cpk_extract.py <cpk路径> 输出目录 --filter cbdata
```

### 注意
- 文件名编码：默认 cp936（简中），日韩版可改 `--codepage 950`
- 路径分隔符：CPK 内部使用 `\`，命令行写双反斜杠 `\\`

---

## 工具二：gdb_parse.py

解析 `*_Softstar.gdb` 二进制数据库为 4 个 JSON 文件，移植自 [GdbFileReader.cs](../../Assets/Scripts/Pal3.Core/DataReader/Gdb/GdbFileReader.cs)。

```bash
python gdb_parse.py <gdb路径> 输出目录 --variant PAL3
python gdb_parse.py <gdb路径> 输出目录 --variant PAL3A
```

输出文件：
- `CombatActors.json` — 战斗角色（含 6 个主角 + 怪物 + Boss）
- `CombatSkills.json` — 所有技能
- `GameItems.json` — 所有物品（武器/防具/药/古董/剧情物品等）
- `CombatComboSkills.json` — 合击技

## 一键重新提取

如果游戏数据更新，可以重新跑一遍：

```bash
cd Notes/tools

# PAL3
python cpk_extract.py "../../Resource/PAL3/basedata/basedata.cpk" --extract-one "cbdata\\PAL3_Softstar.gdb" "../extracted/PAL3/PAL3_Softstar.gdb"
python gdb_parse.py "../extracted/PAL3/PAL3_Softstar.gdb" "../extracted/PAL3" --variant PAL3

# PAL3A
python cpk_extract.py "../../Resource/PAL3A/basedata/basedata.cpk" --extract-one "cbdata\\PAL3A_Softstar.gdb" "../extracted/PAL3A/PAL3A_Softstar.gdb"
python gdb_parse.py "../extracted/PAL3A/PAL3A_Softstar.gdb" "../extracted/PAL3A" --variant PAL3A
```
