## 二、CombatActor（战斗角色）

### 主要字段

| 字段 | 类型 | 含义 |
|------|------|------|
| Id | uint | 角色唯一ID。100-199 是主角ID段，700+ 是怪物 |
| Type | enum | MainActor / Human / Monster / Fairy / God / Ghost / Demon |
| Name | string | 角色名 |
| Level | int | 等级 |
| ElementAttributeValues | dict | 五灵属性值（水火风雷土） |
| AttributeValues | dict | 12项属性（Hp/Sp/Mp/Attack/Defense/Speed/Luck + 五灵） |
| CombatStateImpactTypes | dict | 抗性/弱点：22种战斗状态的免疫/易感 |
| EscapeRate | float | 逃跑成功率（敌方为玩家逃跑成功率） |
| Money | ushort | 金钱（保存用，怪物无意义） |
| MoneyWhenKilled | short | 击杀后掉落金钱 |
| Experience | int | 击杀获得经验 |
| NormalLoot | uint | 普通掉落物品ID |
| NormalLootCount | short | 掉落数量 |
| CorpseId / CorpseSkillId | uint | 尸块ID / 拾取技能ID（炼丹用） |
| CorpseSuccessRate | short | 尸块拾取成功率 |
| StealableItemId / StealableMoneyAmount | uint/short | 偷取物品 / 偷取金钱 |
| SkillIds[4] | uint[] | 角色已学技能ID（4个槽） |
| SkillLevels[4] | byte[] | 对应技能等级 |
| NormalAttackModeId | uint | 普攻动作模式（916-927，区分行/列攻击） |
| SpecialActionId | int | 特殊动作ID |
| HeightLevel | byte | 迷宫高度（0低/1中/2高） |
| MoveRangeLevel / AttackRangeLevel / MoveSpeedLevel / ChaseSpeedLevel | byte | 移动/攻击范围/速度等级 |

## 三、Skill（技能）

### 主要字段

| 字段 | 类型 | 含义 |
|------|------|------|
| Id | uint | 技能ID。前两位通常对应主角ID |
| Type | enum | StandardMagic（标准）/ AssistMagic（辅助）/ DamageMagic（伤害）/ RecoverMagic（恢复）/ WeaponMagic（武器技） |
| Name | string | 技能名 |
| TargetRangeType | enum | 目标范围（单体/全体/一行/一列） |
| ApplicableActors | list | 可学习此技能的主角列表 |
| AttributeImpacts | dict | 对目标 12 项属性的影响（含影响类型 Absolute/Percentage/RecoverToMax/IncreaseMax） |
| CombatStateImpactTypes | dict | 施加/解除的战斗状态 |
| ElementAttributes | list | 技能五灵属性 |
| SpConsumeValue / MpConsumeValue | int | SP/MP 消耗值 |
| SpConsumeImpactType / MpConsumeImpactType | enum | 消耗类型（Absolute/Percentage/RecoverToMax） |
| SuccessRateLevel | byte | 成功率等级（影响异常状态命中率） |
| Level | byte | 当前技能等级 |
| TimesBeforeLevelUp[4] | byte[] | 每级升级所需使用次数 |
| RequiredActorLevel | byte | 学习所需角色等级 |
| MagicLevel | byte | 仙术等级（影响伤害公式） |
| NextLevelSkillId | uint | 升级后的技能ID（链式升级） |
| IsUsableOutsideCombat | bool | 战斗外可用（如治疗） |
| CompositeSkillIds[3] | uint[] | 复合技能ID |
| CanTriggerComboSkill | bool | 是否可触发合击 |

### 技能ID段约定（PAL3 部分）

- 41xx — 雪见技能（飞龙探云手 4101 等）
- 42xx — 龙葵·蓝技能（绿波红露斩 4201 等）
- ...

## 四、GameItem（物品）

### 主要字段

| 字段 | 类型 | 含义 |
|------|------|------|
| Id | uint | 物品ID |
| Name | string | 物品名 |
| Price | int | 价格（购买价；商店出售价格通常是此值的一半） |
| Type | enum | None / Healing / Throwable / Treasure / Antique / Ore / Plot / Cloth / Hat / Shoes / Wearable / Weapon / Corpse / Blueprint(PAL3A) |
| WeaponType | enum | None / Spear / Sword / Staff / Machete / Bow / Spine / Sickle |
| ApplicableActors | list | 可使用/装备的主角列表 |
| ElementAttributes | list | 物品五灵属性 |
| AncientValue | int | 古董价值（典当行用） |
| ItemSpecialType | enum | 特殊效果（Poison/Flee/LevelIncrease/ExIncrease5%/15%） |
| TargetRangeType | enum | 物品作用目标范围 |
| PlaceOfUseType | enum | None / Combat / OutOfCombat / Anywhere |
| AttributeImpacts | dict | 对属性的影响（药物加血/装备加属性） |
| CombatStateImpacts | dict | 战斗状态影响 |

#### PAL3 装备特有字段
- `ComboCount` — 连击次数
- `SpSavingPercentage` / `MpSavingPercentage` — 气/神消耗减少%
- `CriticalAttackAmplifyPercentage` — 暴击伤害增加%
- `SpecialSkillSuccessRate` — 特技成功率增加%
- `OreId` / `ProductId` / `ProductPrice` — 尸块炼丹相关
- `SynthesisMaterialIds[2]` / `SynthesisProductId` — 合成相关

#### PAL3A 装备特有字段
- 同 PAL3 的 SpSaving/MpSaving/CriticalAmp/SpecialSkillSuccessRate
- `CreatorActorId` — 制作角色ID
- `MaterialId` — 制作所需材料
- `ProductType` — 0美食方/梦溪杂录, 1装备图谱/花种, 2武器图谱
- `ProductId` — 制作出的物品ID
- `RequiredFavorValue` — 制作所需最低好感度

## 六、关键枚举速查

### ActorAttributeType（属性类型，索引顺序）
```
0=Hp(精)  1=Sp(气)  2=Mp(神)  3=Attack(攻)  4=Defense(防)
5=Speed(速)  6=Luck(运)  7-11=Water/Fire/Wind/Thunder/Earth(五灵)
```

### ActorCombatStateType（22种战斗状态）
```
0-4    五灵毒（风/雷/水/火/土毒）
5-12   增减益（攻防运速 各 ±10%）
13-21  控制类：定/封/禁/眠/乱/狂/镜/避/界
22-30  其他：隐形/濒死/死亡/避毒/任意毒/辟邪/退魔/神眼/精进
```

### TargetRangeType
```
0=None  1=我方单体  2=我方全体
3=敌方单体  4=敌方全体  5=敌方一排  6=敌方一列
```

### AttributeImpactType
```
0=Absolute(绝对值)  1=Percentage(百分比)
2=RecoverToMax(恢复至上限)  3=IncreaseMax(增加上限)
```

## 七、用于战斗系统的关键观察

1. **属性公式起点**：主角 Lv1 HP 在 60-100 区间，Attack 在 20-35，Defense 在 14-18，Speed 在 3-5。这给战斗数值平衡提供了基线。
2. **技能消耗结构**：技能用 SP（气）和 MP（神）双资源，SP 通常用于武学/特技，MP 用于仙术。
3. **状态系统**：22 种战斗状态需要在战斗回合管理器中实现倒计时、叠加/覆盖、抗性判定。
4. **目标选择**：6 种目标范围决定 UI 选择交互（单体需选目标，全体/排/列直接执行）。
5. **合击触发**：固定参战角色 + 元素位置/武器类型/战斗状态组合判定。

后续战斗系统实现时，所有数值直接从 `extracted/PAL3/` JSON 读取或在 Unity 中通过 `GameResourceProvider.GetGameDatabaseFile()` 获取。
